# Lens Mascot Style Guide

This page previews every Lens mascot pose and admonition style. Lens is the
book's evidence-focused research guide—not a customer persona or a source of
customer evidence.

!!! mascot-neutral "A Note from Lens - Neutral"
    ![Lens in a neutral pose](../img/mascot/neutral.png){ class="mascot-admonition-img" }
    Keep observations, assumptions, and conclusions separate so another
    researcher can follow your reasoning.

!!! mascot-welcome "Let's Look Closer - Welcome"
    ![Lens waving welcome](../img/mascot/welcome.png){ class="mascot-admonition-img" }
    Start with the decision you need to support. Then choose the evidence and
    evaluation method that fit that decision.

!!! mascot-thinking "Key Insight - Thinking"
    ![Lens thinking](../img/mascot/thinking.png){ class="mascot-admonition-img" }
    A plausible persona response is still a model output. What additional
    evidence would you need before making a claim about real customers?

!!! mascot-tip "Lens's Research Tip - Tip"
    ![Lens giving a tip](../img/mascot/tip.png){ class="mascot-admonition-img" }
    Save the asset, prompt, rubric, model settings, and run date together. That
    small habit makes later comparisons much easier to interpret.

!!! mascot-warning "Pause and Check - Warning"
    ![Lens giving a warning](../img/mascot/warning.png){ class="mascot-admonition-img" }
    Do not present synthetic reactions as customer evidence. Use them to form
    hypotheses that appropriate research with real people can examine.

!!! mascot-encourage "Keep Investigating - Encouraging"
    ![Lens offering encouragement](../img/mascot/encouraging.png){ class="mascot-admonition-img" }
    A failed prompt test is useful evidence. Record the failure, revise one
    variable, and run the comparison again.

!!! mascot-celebration "Evidence Secured - Celebration"
    ![Lens celebrating](../img/mascot/celebration.png){ class="mascot-admonition-img" }
    You connected the recommendation to its evidence and documented the
    uncertainty. That is meaningful research progress.
