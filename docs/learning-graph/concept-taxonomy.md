# Concept Taxonomy

The 400 concepts are organized into 14 primary categories.

## AI And Research Foundations

**TaxonomyID:** `AIFND`

Core AI, language model, synthetic user, and customer research concepts. This category contains concepts 1–20.

## Research Design And AI Limits

**TaxonomyID:** `RSRCH`

Research methods, evidence quality, model limitations, and bias awareness. This category contains concepts 21–60.

## Ethics Privacy And Governance

**TaxonomyID:** `ETHIC`

Responsible use, privacy, fairness, transparency, oversight, and accountability. This category contains concepts 61–80.

## Persona Design

**TaxonomyID:** `PERDS`

Persona attributes, motivations, needs, behaviors, scenarios, and evidence bases. This category contains concepts 81–120.

## Persona Segmentation And Quality

**TaxonomyID:** `PERQ`

Segmentation, Jobs-to-be-Done, journeys, persona testing, consistency, and reuse. This category contains concepts 121–180.

## Prompt Design And Outputs

**TaxonomyID:** `PRMPT`

Prompt structure, context, roles, constraints, templates, and structured responses. This category contains concepts 181–220.

## Prompt Testing And Dialogue

**TaxonomyID:** `PTEST`

Prompt evaluation, failure modes, question design, and synthesis techniques. This category contains concepts 221–240.

## Agent Design And Roles

**TaxonomyID:** `AGENT`

Agent anatomy, boundaries, persona roles, and specialist reviewer roles. This category contains concepts 241–260.

## Multi-Agent Orchestration

**TaxonomyID:** `MULTI`

Coordination, discussion, debate, aggregation, consensus, and critique patterns. This category contains concepts 261–290.

## Brand And Asset Evaluation

**TaxonomyID:** `BRAND`

Brand strategy, messaging, identity, channels, assets, and customer journeys. This category contains concepts 291–320.

## Rubrics Scores And Evidence

**TaxonomyID:** `RUBRC`

Evaluation frameworks, scoring, evidence, confidence, and marketing criteria. This category contains concepts 321–350.

## Analysis And Recommendations

**TaxonomyID:** `ANLYS`

Pattern analysis, risk assessment, prioritization, and iterative improvement. This category contains concepts 351–370.

## Graph-Based Knowledge Models

**TaxonomyID:** `GRAPH`

Knowledge graph structures, relationships, evidence, and organizational memory. This category contains concepts 371–386.

## Automation Reporting And Capstone

**TaxonomyID:** `WORKF`

No-code workflows, automated reports, dashboards, and scalable systems. This category contains concepts 387–400.
